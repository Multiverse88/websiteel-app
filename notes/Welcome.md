# Welcome to EasyLegal Vault

Knowledge base untuk project websiteel-app.

## Cara Pakai

- `Cmd/Ctrl + O` — cari catatan
- `[[wikilink]]` — link antar catatan
- `#tag` — tagging
- Folder di kiri → browse

## Struktur

- `devlog/` — catatan harian progress
- `notes/` — catatan bebas
- `ideas/` — ide fitur
- `docs/` — dokumentasi teknis

---

Vault ini terintegrasi dengan opencode — semua catatan bisa dibaca/ditulis langsung dari terminal.
